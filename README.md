# odin-recipes

A recipe website built as part of The Odin Project's HTML Basics course. The goal was structure, not styling — pure HTML, no CSS yet.

🔗 **[Live site](https://chris-jacob-dela-cerna.github.io/odin-recipes/)**

---

## What this is

This project is about learning how HTML actually organizes content — headings, lists, images, semantic structure — before any visual design comes into play. Everything here is unstyled on purpose.

---

## What this is not

This isn't a finished recipe site. It's the HTML skeleton TOP asks for at this stage. Styling, and possibly more recipes, may come later if I return to it — but as of now, this reflects exactly where I am in the course.
